# TaskBloom ✨

TaskBloom is a modern productivity and task management web application designed with a dreamy glassmorphism interface, smooth animations, and an immersive user experience.

The project combines elegant UI design with practical productivity features like task tracking, progress analytics, notes management, and local storage persistence.

Built using React, TypeScript, Vite, JavaScript, and CSS, TaskBloom focuses on creating a calm and aesthetic environment for managing daily tasks and goals.

---

## 🌟 Features

- ✅ Add, complete, and delete tasks
- 📊 Progress tracking dashboard
- 📝 Notes management system
- ✨ Beautiful glassmorphism UI
- 🌌 Animated backgrounds and floating particles
- 📈 Weekly analytics visualization
- 💾 localStorage data persistence
- 🔔 Interactive success popups
- 🎵 Sound effects using Web Audio API
- 📱 Fully responsive design

## ⚡ Core Functionalities

### ✅ Task Management

Users can:

- Add tasks
- Mark tasks as completed
- Delete tasks
- Track pending and completed work

### 📊 Analytics Dashboard

The app dynamically updates:

- Total tasks
- Completed tasks
- Pending tasks
- Progress percentage
- Weekly activity graph

### 💾 Local Storage

All tasks and notes are stored in the browser using localStorage, allowing data to remain available after page refresh.
